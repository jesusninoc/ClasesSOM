# Ejecutar los comandos que están dentro de un fichero
PowerShell
```PowerShell
"ps" > cosasejecutar.txt
"date" >> cosasejecutar.txt
cat .\cosasejecutar.txt | %{.$_}
```
Bash (mala soolución)
```Bash
echo "ps" > fichero
echo "date" >> fichero
nano mifichero.sh
for fn in `cat fichero`; do
    $fn
done
./mifichero.sh
```
