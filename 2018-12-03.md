# Ejecutar los comandos que están dentro de un fichero
PowerShell
```PowerShell
"ps" > cosasejecutar.txt
"date" >> cosasejecutar.txt
cat .\cosasejecutar.txt | %{.$_}
```
Bash (solución 1)
```Bash
echo "ps" > ejecutar
echo "date" >> ejecutar
cat ejecutar | awk '{print $1}' | sh
```
Bash (solución 2)
```Bash
echo "ps" > ejecutar
echo "date" >> ejecutar
cat ejecutar | cut -f1 | sh
```
