# Ejecutar los comandos que están dentro de un fichero
PowerShell
```PowerShell
"ps" > cosasejecutar.txt
"date" >> cosasejecutar.txt
cat .\cosasejecutar.txt | %{.$_}
```
Bash (buena solución)
```Bash
echo "ps" > ejecutar
echo "date" >> ejecutar
cat ejecutar | awk '{print $1}' | sh
```
Bash (mala solución)
```Bash
echo "ps" > ejecutar
echo "date" >> ejecutar
`cat ejecutar | awk '{print $1}'`
```
