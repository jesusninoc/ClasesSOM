# Copy
# Move

## Ejemplos
```PowerShell
Move-Item .\examen -Destination .\examen2
Get-ChildItem .\examen2

Copy-Item .\examen2 .\examen3

ls examen?

mkdir (1..5)

rm '.\1 2 3 4 5'

ls ? | %{
    copy .\examen.txt $_
}
```

## Ejercicio: copiar un fichero en varias carpetas con PowerShell
```PowerShell
foreach ($elementos in mkdir (1..5)){cp .\examen.txt $elementos}
```
