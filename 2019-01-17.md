# Script en Linux que suma dos número que introduce el usuario
```Bash

read num2

```
# Script en Linux que suma dos números como parámetros de entrada
Script suma (crear con nano)
```Bash
num=$(($1 + $2))
echo $num
```
Llamar al script suma
